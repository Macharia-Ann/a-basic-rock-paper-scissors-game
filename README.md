# Rock Paper Scissors

A simple Rock Paper Scissors game built with HTML, CSS, and Vanilla JavaScript.

## Features

- Play against the computer
- Random computer choices
- Dynamic result display
- Interactive UI
- End game button

## Concepts Practiced

- DOM manipulation
- Event listeners
- Functions
- Conditional logic
- `Math.random()`
- Dynamic HTML rendering

## Technologies Used

- HTML5
- CSS3
- JavaScript
